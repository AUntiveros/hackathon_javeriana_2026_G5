# 📚 Investigación — Nino (Alzheimer, LATAM)

Carpeta que centraliza toda la investigación que respalda el diseño. Deliverables obligatorios del
hackathon + soporte para el pitch.

## Delimitación oficial del problema (leer primero)

> **Problema:** el olvido de una **actividad cotidiana previamente programada**.
> **Alcance:** sistema de recordatorios para (al menos) **una rutina** (medicamento, cita, comida,
> autocuidado), que **confirma si se realizó** y **notifica al cuidador si no**.
> **Prohibido:** diagnosticar Alzheimer, evaluar progresión cognitiva, controlar TODAS las
> actividades del paciente.
